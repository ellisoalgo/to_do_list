
# To Do List

A simple To Do List app that allows users to add tasks, delete them, and filter by name to search for specific tasks.

## Technologies

- React
- Vite
- HTML
- CSS
- JavaScript

## Installation

Install to_do_list with npm:

```bash
  git clone https://github.com/ellisoalgo/to_do_list.git
  cd to_do_list
  npm install
```

To run:

```bash
  npm run dev
```
